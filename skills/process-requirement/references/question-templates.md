# 按领域分类的提问模板

根据工作类型使用以下模板来引导需求澄清。

## 前端 / UI 功能

- 涉及哪些页面或组件？
- 是否有设计稿或线框图？
- 响应式/断点要求是什么？
- 是否有无障碍（a11y）要求？
- 加载中/错误/空状态分别如何处理？
- 是否需要动画或过渡效果？

## 接口 / 后端功能

- 需要创建或修改哪些接口？
- 请求/响应的数据结构是什么？
- 需要什么认证/授权机制？
- 是否有限流或节流要求？
- 错误信息如何返回给客户端？
- 是否需要数据库迁移？

## 数据库变更

- 涉及哪些表/集合？
- 是否是破坏性的结构变更？
- 现有数据是否需要迁移？
- 需要哪些索引来保证性能？
- 是否有外键或约束变更？

## 集成 / 第三方服务

- 涉及哪个外部服务或接口？
- 是否有沙箱/测试环境可用？
- 重试和超时策略是什么？
- 外部服务故障时如何处理？
- 是否有 Webhook 或回调需求？

## 缺陷修复

- 预期行为和实际行为分别是什么？
- 能否提供复现步骤？
- 哪些环境受到影响（开发/预发/生产）？
- 目前是否有临时解决方案？
- 严重程度和紧急程度如何？

## 重构

- 重构的动机是什么？
- 外部行为是否需要保持不变？
- 当前行为是否有测试覆盖？
- 可接受的变更范围是什么？
- 是否有需要维持的性能基准？
